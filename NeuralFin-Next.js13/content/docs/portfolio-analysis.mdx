# Portfolio Analysis

NeuralFin provides powerful tools for portfolio analysis, helping you understand your investments' performance, risk, and potential returns. This guide will walk you through the key features and how to use them effectively.

## Key Features

### Portfolio Performance
- Total return calculation
- Risk-adjusted returns (Sharpe ratio, Sortino ratio)
- Performance attribution
- Benchmark comparison

### Risk Analysis
- Value at Risk (VaR)
- Expected Shortfall
- Portfolio volatility
- Correlation analysis
- Sector exposure

### Portfolio Optimization
- Modern Portfolio Theory (MPT)
- Risk parity optimization
- Factor-based optimization
- Rebalancing recommendations

## Getting Started

First, ensure you have the NeuralFin client library installed:

```bash
npm install @neuralfin/client
```

Initialize the client with your API key:

```js
import { NeuralFinClient } from '@neuralfin/client';

const client = new NeuralFinClient({
  apiKey: 'your-api-key'
});
```

## Examples

### Analyze Portfolio Performance

```js
const portfolio = {
  holdings: [
    { symbol: 'AAPL', shares: 100 },
    { symbol: 'GOOGL', shares: 50 },
    { symbol: 'MSFT', shares: 75 }
  ]
};

const analysis = await client.analyzePortfolio(portfolio);
console.log(analysis);
// {
//   totalValue: 45000.00,
//   dailyReturn: 0.015,
//   monthlyReturn: 0.045,
//   yearlyReturn: 0.18,
//   volatility: 0.15,
//   sharpeRatio: 1.2
// }
```

### Risk Analysis

```js
const riskMetrics = await client.getPortfolioRisk(portfolio);
console.log(riskMetrics);
// {
//   var95: 0.02,
//   expectedShortfall: 0.03,
//   beta: 1.1,
//   correlation: 0.85
// }
```

### Portfolio Optimization

```js
const optimization = await client.optimizePortfolio({
  symbols: ['AAPL', 'GOOGL', 'MSFT', 'AMZN', 'META'],
  constraints: {
    maxWeight: 0.3,
    minWeight: 0.05
  }
});
console.log(optimization);
```

## Best Practices

1. **Regular Analysis**: Perform portfolio analysis regularly to track performance and risk.
2. **Diversification**: Ensure your portfolio is properly diversified across sectors and asset classes.
3. **Rebalancing**: Rebalance your portfolio periodically to maintain target allocations.
4. **Risk Management**: Monitor risk metrics and adjust positions as needed.
