# Integrations

Connect NeuralFin to your favorite tools and platforms.

## Supported Integrations

- Google Sheets
- Zapier

## Example: Use NeuralFin in Excel

1. Install the NeuralFin Excel Add-in
2. Use the `=NEURALFIN.PREDICT("AAPL", 7)` formula in your spreadsheet
