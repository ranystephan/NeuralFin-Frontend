# Market Data

NeuralFin provides access to comprehensive real-time and historical market data through our powerful API. This guide will help you understand the available data types and how to use them effectively.

## Available Data Types

### Stock Prices
- Real-time quotes
- Historical daily prices
- Intraday data (1m, 5m, 15m, 30m, 1h intervals)
- Adjusted and unadjusted prices
- Volume and trading statistics

### Market Indices
- Major global indices (S&P 500, NASDAQ, etc.)
- Sector-specific indices
- Custom index calculations

### Economic Indicators
- GDP data
- Inflation rates
- Interest rates
- Employment statistics
- Other macroeconomic indicators

## Getting Started

First, ensure you have the NeuralFin client library installed:

```bash
npm install @neuralfin/client
```

Then initialize the client with your API key:

```js
import { NeuralFinClient } from '@neuralfin/client';

const client = new NeuralFinClient({
  apiKey: 'your-api-key'
});
```

## Examples

### Get Real-Time Quote

```js
const quote = await client.getQuote('AAPL');
console.log(quote);
// {
//   symbol: 'AAPL',
//   price: 175.04,
//   change: 1.23,
//   changePercent: 0.71,
//   volume: 52345678,
//   timestamp: '2024-03-20T14:30:00Z'
// }
```

### Get Historical Prices

```js
const data = await client.getHistoricalPrices({
  symbol: 'AAPL',
  start: '2023-01-01',
  end: '2023-06-01',
  interval: '1d'  // daily data
});
console.log(data);
```

### Get Market Indices

```js
const indices = await client.getIndices(['^GSPC', '^IXIC']);
console.log(indices);
```

## Best Practices

1. **Rate Limiting**: Be mindful of API rate limits. Use appropriate caching strategies for frequently accessed data.
2. **Error Handling**: Always implement proper error handling for API requests.
3. **Data Validation**: Validate the data before using it in your application.
